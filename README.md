# 🎓 Student Submission Tracker

## 📌 Description
The Student Submission Tracker is a .NET-based application developed to manage and track student assignment submissions efficiently. It helps faculty monitor submission status, deadlines, and student performance.

---

## 🚀 Features
- Add and manage assignments  
- Track student submissions  
- Deadline management  
- Late submission detection  
- Faculty & student login  

---

## 🛠️ Technologies Used
- .NET (Windows Forms)
- C#
- SQL Server

---

## ▶️ How to Run
1. Open project in Visual Studio  
2. Connect database  
3. Run the project
